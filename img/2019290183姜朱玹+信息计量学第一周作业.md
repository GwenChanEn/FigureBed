(1)什么叫做“信息计量学”，其研究对象和基本内容是什么? 

```
		简单地说，信息计量学就是，采用定量的方法来描述和研究信息的现象、过程和规律的一门学科。
		信息计量学的研究对象目前主要是**各种事物信息的数量方面**。其内容 包括消息、数据、事件、实物、文本和文献等，其中有正式交流的事物信息，也有非正式交流的事物信息。
		基本研究内容如下：
		• 通过信息计量学的理论研究，解决研究过程中的方法论问题;
    • 通过对科学活动中的信息过程和信息现象的研究，发现和揭示信息存的方式及其分布的基本规律;
    • 实现文献信息管理的科学化，为信息服务和信息管理决策提供量化依据。
    基本内容体系：
    从“狭义信息计量学”角度展开，其内容体系一般包括:(1)理论(基本问题、理论基础、基本规律、数据准备) (2)方法论 (3)应用领域
    ❶信息计量学基本理论研究具体包括，信息计量学研究对象、信息的数学描述、研究内容与理论体系，以及与其它学科的关系等。
    ❷信息的基本测度具体包括，确定信息计量的基本单位，例如:比特、知识单元、信息熵、信息场、信息势等。
    ❸信息流模型的研究
    具体包括， 文献增长、老化、离散、引文分析等模型的建立与评价。
    ❹对几个基本定律的研究。特别是在当前网络环境和数据环境下的适用性研究。
    ❺信息计量方法和工具的研究
    如:信息聚类、相关分析、词频统计、共词分析、可视化呈现等。特别是当前网络数据化环境下，信息计量的方法。
    ❻信息计量的相关应用研究
    如:在图书情报工作中的应用研究、在科学评价中的应用、在信息分 析中的应用等。
```

### **2.** 讨论题

(1)信息计量学的产生背景是什么?研究意义何在? 

```
		信息计量学原称情报计量学，最早由德国学者昂托.纳克教授(Otto Nacke)提出。
		研究意义
		理论方面:检验规律的应用广度和深度(信息单元、知识单元和网络信息);实际应用方面:利用研究方法和计量技术，科学管理信息，优化馆藏，预测科技事业的发展及其对经济、社会的影响，可以评价二次文献的完整性，提高信息检索的查全率和查准率，评价人才和机构等。
```



(2)信息计量学的主要应用领域有哪些?

```
		首先，信息计量学与许多学科有着一定的联系。关系较为密切的有:图书馆学、文献学、情报学、数学、 统计学、文献计量学、科学计量学、科学评价、网络信息计量等。重要的应用之一是研究学术传播 (scholarly communication) 的模式，例如学术成果的定量评价、学术创新的历史传播、学术传播的结构分析等等。
		互联网作为一个强大的传播媒介正在引起信息生产和交换方式的变革，随着各种信息在互联网上的大量涌现、学术传播系统从以学术期刊为中心的模式向网络化新模式的发展，信息计量学的理论和方法一方面正在被成功地应用于开发增值信息服务和提高信息组织和检索的效率，另一方面也面临着新的数据和工具给其进一步发展和丰富而带来的机会和挑战。本章旨在探讨这些方面的问题，首先概述信息计量学在网络环境中的信息组织与检索方面的应用，然后探讨网络环境中的学术传播系统给信息计量学的进一步发展所带来的机会，最后介绍网络计量学 (Webometrics) 的主要研究分支和目前研究中所面临的一些问题。
```
